## Day 1 Problem

একটা `function` লিখো, Function এর নাম হবে:

```js
findLargestNumber;
```

Function একটি array of numbers parameter হিসেবে নিবে। তোমাকে array থেকে সবচেয়ে বড় number return করতে হবে।

Example:

```md
Input:
[5, 1, 9, 3, 7]

Output:
9
```

###

আরেকটা example

```md
Input:
[10, 22, 8, 45, 3]

Output:
45
```

#

> [!NOTE]
> তুমি ব্যবহার করতে পারবে না `Math.max()`, তোমাকে নিজের logic দিয়ে বের করতে হবে।

Function structure:

```js
function findLargestNumber(numbers) {
  // body
}
```

#

> [!TIP]
> _যদি array empty হয় `[]` তখন return করবে `"Array is empty"`_
