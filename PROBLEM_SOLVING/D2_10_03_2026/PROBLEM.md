## Day 2 Problem

একটা function লিখতে হবে। Function name: `reverseString`, এই function একটি string parameter নিবে। তোমাকে সেই string উল্টে (reverse) করে return করতে হবে।

```md
Example 1:

input: "hello" output: "olleh"

Example 2:

input: "javascript" output: "tpircsavaj"
```

##

> [!NOTE]
> তুমি ব্যবহার করতে পারবে না: `reverse()`, `split()`, `join()`
>
> মানে built-in shortcut না। নিজের logic ব্যবহার করতে হবে।

##

Function structure

```js
function reverseString(str) {
  // body
}
```

Edge Case: Input যদি হয় `""` তাহলে return করবে `""`
