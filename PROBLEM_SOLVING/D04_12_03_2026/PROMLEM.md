## Day 4 Problem

একটা function লিখতে হবে। Function name `countProperties`. এই function একটি object parameter হিসেবে নিবে। তোমাকে বের করতে হবে object এর ভিতরে কয়টা property আছে এবং সেই সংখ্যা return করতে হবে।

```js
/**
 * Example 1
 *
 * input
*/
{
  name: "Shahed",
  age: 25,
  city: "Dhaka"
}
// Output: 3


/**
 * Example 2
 *
 * input
*/
{
  a: 10,
  b: 20
}
// output 2
```

##

> [!NOTE]
> তুমি ব্যবহার করতে পারবে না: `Object.keys()`, `Object.values()`, `Object.entries()`
>
> মানে built-in shortcut না। নিজের logic ব্যবহার করতে হবে।

##

```js
// Function structure
function countProperties(obj) {
  // body
}

/**
 * Edge Case
 * Input যদি হয়: {}, তাহলে return করবে: 0
 */
```
